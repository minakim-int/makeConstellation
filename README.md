# makeConstellation
OpenFrameWorksで制作した「星座づくり」プログラムです。

## あそび方
* 星をマウスで選択して、繋げて行きます。
* 選択した星を再びクリックすると線が伸びなくなります。
* 思った通りに出来なかった場合、Resetボタンを押してやり直せます。
  ** 「取り消し」機能追加予定
* 完成したらSaveボタンで/bin/dataに現在の画像が保存されます。
